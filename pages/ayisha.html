Ayisha — Tutorial componenti (code + live demo)

Ogni sezione mostra il codice sorgente del componente e una demo statica che ne rappresenta l'aspetto. Questi esempi sono statici e servono a mostrare markup e binding tipici usati con Ayisha.

Step 1: Getting started con @model

Creiamo un semplice campo input legato ai dati con la direttiva @model.

Codice:

<div class="todo-app-container">
    <div class="todo-header">
        <h2>My Todo App</h2>
        <p>Built with Ayisha.js</p>
    </div>
    <div class="todo-input-section">
        <input type="text" @model="newTodo" class="todo-input" placeholder="What needs to be done?">
    </div>
    <p>You typed: <span @text="newTodo"></span></p>
</div>

Header (component: components/header.html)

Mostra le varianti via @switch/@case e il toggle per dispositivi mobili.

Codice:

<span @switch="_currentPage" @set="headerExpanded = true">
    <header @default class="header">...</header>
    <header @case="categories" class="header">...</header>
    <header @case="search" class="header">...</header>
    <header @case="random" class="header">...</header>
</span>

Nav & Links (components/nav.html, components/links.html)

Codice (links.html):

<a href="#" @link="home" @class="{'nav-link': true, 'active': _currentPage === 'home'}">⌂ Home</a>
<a href="#" @link="categories" @class="{'nav-link': true, 'active': _currentPage === 'categories'}">⊞ Categories</a>
<a href="#" @link="search" @class="{'nav-link': true, 'active': _currentPage === 'search'}">⊙ Search</a>
<a href="#" @link="random" @class="{'nav-link': true, 'active': _currentPage === 'random'}">⊛ Random</a>

Search Section (components/search-section.html)

Codice:

<div class="search-section">
    <h2>Welcome to the world of recipes!</h2>
    <p>Explore thousands of recipes...</p>
    <div class="search-container">
        <input type="text" @model="quickSearch" class="search-input" placeholder="Search for a recipe...">
        <button @click="searchQuery = quickSearch; currentPage = 'search'; quickSearch = ''" @link="search" class="search-btn">⊙ Search</button>
    </div>
</div>
